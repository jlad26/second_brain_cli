# Installation

## Prerequisites

Python >= 3.12 installed.

Python Installation via pyenv:

1. Install pyenv - instructions [here](https://github.com/pyenv/pyenv)
1. Set python version to global access: `pyenv global 3.12`

pipx installation
1. Install pipx: `pip install --user pipx`
1. Restart terminal, then: `pipx ensurepath`

## CLI installation

`pipx install git+https://github.com/jlad26/second_brain_cli.git`
